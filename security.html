Security

Keeping Farming Ledger dependable.

Security is handled through HTTPS delivery, account controls, hosted app infrastructure, careful downloads, and clear reporting.

Website and app transport

FarmingLedger.co.za is intended to run over HTTPS so data is encrypted between your browser and the website or online app.

App data and access

The app is Supabase-backed at a high level, using managed authentication and database services. Account access should be protected with strong passwords and careful device management.

Downloads and updates

Install Farming Ledger only from the official website download links. Updates may replace app files, improve reliability, and address security issues.

Your responsibilities

Keep your device updated, protect your login details, sign out on shared devices, and avoid uploading information you are not authorized to store.

Report a security issue

If you find a security issue, email hello@FarmingLedger.co.za with the affected page, device, browser, and steps to reproduce the issue.
